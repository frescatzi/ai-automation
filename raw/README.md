# raw/ — sources curées & immuables

**Tu déposes ici** les sources que tu as choisies (exports Claude/ChatGPT, articles, refs, papiers).

Règles :
- **Immuable** : on **ajoute**, on ne réécrit jamais. Provenance préservée.
- Le **LLM lit** ces fichiers, ne les **modifie jamais**.
- Nommage : `AAAA-MM-JJ--source-titre-court.md`.
- Frontmatter `type: raw` (voir `../CLAUDE.md § 3.2`).

> Ce n'est pas une inbox en vrac : `raw/` est curé, `wiki/` est propre.
