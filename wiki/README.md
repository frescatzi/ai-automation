# wiki/ - pages propres (écrites par le LLM)

**Le LLM écrit ici** : résumés, pages-concepts, synthèses. **Tu lis.**

Règles :
- Une page = un concept ou une synthèse. Nommage : `concept-nom.md` / `synthese-sujet.md`.
- Frontmatter `type: wiki` avec `status` et `publish` (voir `../CLAUDE.md § 3.3`).
- Backlinks `[[…]]` internes au coffre. Les renvois hors coffre se font en texte.
- Une page n'atteint les agents que si `status: active` **et** `publish: notion`.
